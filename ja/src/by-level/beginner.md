# 初級

まず作りたいものを選んでください。それぞれの初級ガイドは、その題材を通じて Rust の基本的な読み書きができるレベルを目標にしています。

## ガイドを選ぶ

| ガイド | 題材 | 目安 |
| --- | --- | --- |
| [Web バックエンド 初級](../web-backend/beginner.md) | axum を使った Web サーバーの good first issue を解く | 約 2 時間 |
| [CLI ツール 初級](../cli/beginner.md) | ファイルの行数・文字数を表示する CLI ツールのバグを修正する | 約 2 時間 |
| [WASM 初級](../wasm/beginner.md) | ブラウザで動く diff ツールのバグを修正する | 約 3 時間 |
| [低レイヤ 初級](../low-level/beginner.md) | ファイルのパーミッション表示ツールのバグを修正する | 約 2 時間 |
| [組み込み 初級](../embedded/beginner.md) | micro:bit の LED やセンサを動かすプログラムを育てる | 約 3 時間 |

目安はページ数と文字数から算出しています。コードを手元で動かしながら読む場合の時間です。

どれにするか迷ったら [Web バックエンド 初級](../web-backend/beginner.md) から始めてください。Web 開発の経験がなくても読み進められます。
