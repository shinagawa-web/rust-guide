# レベル別ガイド

Rust をどれくらい触ったかで、始める場所を選んでください。

- [初級](beginner.md) は、Rust に初めて触れる人向けです。動くコードを修正しながら、基本的な構文と考え方を身につけます。
- [中級](intermediate.md) は、初級を終えた人向けです。機能を加えてテストを書き、配布またはデプロイできる状態まで仕上げます。
- 上級は準備中です。

どちらか迷ったら [初級](beginner.md) から始めてください。

TypeScript・Go・C 言語をすでに知っている場合は、[言語別ガイド](../by-language/) から入ることもできます。
