# 中級

初級で作ったものを、育てて仕上げます。それぞれの中級ガイドは、初級の題材を起点に機能を加え、配布またはデプロイできる状態まで持っていくことを目標にしています。

## ガイドを選ぶ

| ガイド | 題材 | 目安 |
| --- | --- | --- |
| [CLI ツール 中級](../cli/intermediate.md) | `rwc` にオプション・エラー処理・テストを加えて配布する | 約 4 時間 |
| [WASM 中級](../wasm/intermediate.md) | ブラウザで動く diff ツールを Cloudflare Workers のエッジ API として公開する | 約 3 時間 |
| [組み込み 中級](../embedded/intermediate.md) | BLE で micro:bit の気温をスマホに届け、LED をスマホから操作する | 約 3 時間 |

目安はページ数と文字数から算出しています。コードを手元で動かしながら読む場合の時間です。

CLI ツールと WASM は専用のスターターリポジトリを起点にするため、対応する初級ルートを経なくても Rust と cargo が使えれば始められます。組み込みは初級の続きとして設計されているため、[組み込み 初級](../embedded/beginner.md) を先に済ませてください。

どれにするか迷ったら [CLI ツール 中級](../cli/intermediate.md) から始めてください。Cloudflare アカウントや実機が不要で、手元だけで完結します。
