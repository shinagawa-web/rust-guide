# このガイドについて

プログラミング経験がある人を対象に、目的や背景に合った入口から Rust を学べるよう構成しています。

## 想定読者

何らかのプログラミング経験があり、これから Rust を学ぶ人を想定しています。特定の言語や前提知識を求めるものではなく、必要なことはこのガイドの中で説明します。

なぜ既存のドキュメントと別にこのガイドを作ったのか、その問題意識は [なぜこのガイドを作ったか](why-this-guide.md) にまとめています。

## 3つの入口

同じ概念でも、読者のバックグラウンドや目的によって「最初に学ぶべき順序」は違います。このガイドは1本道のロードマップを押し付けず、自分に合った入口から読み始められるよう、コンテンツを3つの軸で整理しています。

| 入口 | こんな人に |
|---|---|
| 作りたいもの別（[Web バックエンド](../web-backend/)・[CLI](../cli/) など） | 目的が先にある人 |
| [レベル別](../by-level/) | 初級・中級・上級から、自分の現場や目的に必要なレベルを選んで学びたい人 |
| [言語別](../by-language/) | Go・Python・TypeScript など、使ってきた言語に置き換えて説明されると理解が進む人 |

どの入口を選ぶか迷ったら、[3つの入口の使い分け](how-to-read.md) を参照してください。

## コードを動かしながら読む

コード例には極力 [Rust Playground](https://play.rust-lang.org/) へのリンクを付けています。ローカル環境を用意しなくても、ブラウザ上でそのまま実行して確認できます。詳しくは [Rust Playground の使い方](playground.md) を参照してください。
