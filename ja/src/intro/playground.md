# Rust Playground の使い方

[Rust Playground](https://play.rust-lang.org/) は、ブラウザだけで Rust のコードを書いて実行できる公式の実行環境です。このガイドのコード例に付けたリンクをたどると、その例を読み込んだ状態でこの Playground が開きます。ここでは、そのリンクをただ眺めて終わらせず、自分で手を動かしながら読み進めるための使い方を説明します。インストールやプロジェクト作成の話は出てきません。ブラウザのタブを一枚開くだけで足ります。

## まず動かしてみる

話を進める前に、実際に一つ開いてみましょう。次のリンクが、このガイドのコード例に付くリンクとまったく同じものです。

```rust
fn main() {
    let name = "Rust";
    println!("Hello, {name}!");
}
```

<a href="https://play.rust-lang.org/?version=stable&amp;mode=debug&amp;edition=2021&amp;code=fn%20main%28%29%20%7B%0A%20%20%20%20let%20name%20%3D%20%22Rust%22%3B%0A%20%20%20%20println%21%28%22Hello%2C%20%7Bname%7D%21%22%29%3B%0A%7D%0A" target="_blank" rel="noopener noreferrer">▶ この例を Rust Playground で開く</a>

リンクを開くと、上のコードが読み込まれた状態で Playground が立ち上がります。画面はおおまかに、上部のツールバーと、その下のコードエディタ、実行結果が出る下部の領域に分かれています。コードが表示された状態で実行ボタン（Run）を押すと、サーバー側でコンパイルと実行が走り、結果が下に表示されます。

ここで覚えておきたいのは、エディタはただの表示ではなく、その場で書き換えられるという点です。たとえば先ほどの例なら `name` を自分の名前に変えて Run を押し直せば、出力がすぐ変わるのを確かめられます。コンパイルが通らないコードをわざと書いて、どんなエラーが出るかを見るのも有効な学び方です。Rust はコンパイラのエラーメッセージが丁寧なことで知られています。「なぜこれは弾かれるのか」を知りたくなったら、エディタ上で一行壊してみて、コンパイラに語らせるのがいちばん早い場合があります。

> 📝 実行結果は Standard Output と Standard Error の2つに分かれて出ます。`Hello, Rust!` のようなプログラム本来の出力は Standard Output に、`Compiling ...` や `Running ...` といったビルドの進行ログは Standard Error に出ます。後者はエラーではありません。Cargo は進行状況やコンパイラの警告・エラーを標準エラー出力に流す決まりなので、Standard Error に文字が出ていても、それだけで失敗とは限りません。

## このガイドのリンクの読み方

コード例の近くにある Playground へのリンクは、その例のコードを丸ごと含んだ共有リンクになっています。クリックするとコードが読み込まれた状態で開くので、そのまま実行すれば本文で説明している挙動を再現できます。

リンク先のコードはあなたの手元のセッションに読み込まれるだけなので、いくら書き換えても元のリンクや他の人の画面には影響しません。本文の例を起点にして、変数を足す、型を変える、わざとエラーにするといった寄り道を遠慮なくしてください。元に戻したくなったら、リンクをもう一度開けば最初の状態に戻ります。

自分で試したコードを残したり誰かに見せたりしたいときは、ツールバーの Share ボタンを押すと、その時点のコードが保存され、共有用のリンクが手に入ります。コードを固定した状態で開ける permalink のほか、保存先となる Gist への直接リンクなども表示されます。この保存は裏側で匿名の Gist として作られるので、GitHub にログインする必要はありません。

## ここから先へ

このガイドのコード例を動かすぶんには、ここまでで足ります。

一つだけ覚えておくとよいのは、Playground がブラウザ内の手軽な実行環境だという点です。書いたコードは保存されず（残したいときは前述の Share を使います）、外部との通信もできません。こうした制約の理由や、外部クレートを使う例を動かす方法、エディションの切り替え、コンパイラの内部表現を覗く機能といった一歩進んだ使い方は [Rust Playground を使い込む](playground-advanced.md) にまとめています。いま読む必要はなく、実際にそういう場面に出会ったときに開けば十分です。
