# ここまでとこの先

## ここまでやったこと

最初のページでは、Rust のコードを micro:bit に書き込んでハートを光らせました。そこから少しずつ手を加えて、ボタンや加速度センサに反応するプログラムに育てました。

各ページでやったこと:

- ハートのパターンを `[[u8; 5]; 5]` の格子で表し、LED に表示した
- `is_low()` でボタンが押されているかを読み、押されたボタンに応じて絵を変えた
- 加速度センサから X 軸の値を読み、傾きに応じて絵を変えた

この一連の作業を通じて、`#![no_std]` と `#![no_main]` — OS も標準ライブラリも無い状態で Rust が動くことを体感できたはずです。

## この先やってみること

ここで使った micro:bit v2 には、まだ触れていない機能がたくさん残っています。

温度センサ — チップ内蔵の温度センサで気温を読めます。手で握ると値が上がるので、温度を閾値と比べて絵を変えるだけでも動くものができます。

マイク・スピーカー — v2 にはマイクとスピーカーが載っています。音に反応して LED を動かしたり、音を鳴らしたりできます。

無線通信 — nRF52833 の BLE やシンプルなラジオ通信を使うと、2枚のボードで通信できます。

アニメーション — loop の中でパターンを時間ごとに切り替えると、LED をアニメーションさせられます。

RTIC — より本格的な組み込みプログラムには、割り込みとタスクを管理するフレームワーク RTIC があります。loop で回すポーリング方式から、割り込み駆動の構造に移れます。

どれも、このページまでのコードをベースに少しずつ追加していけます。

## 中級編に進む

BLE（Bluetooth Low Energy）を使ってスマホと繋げる内容を [中級編](../intermediate.md) で扱っています。micro:bit の温度センサが拾った気温をスマホのブラウザにリアルタイムで届け、スマホのボタンで LED を切り替えます。アプリのインストールは不要です。
