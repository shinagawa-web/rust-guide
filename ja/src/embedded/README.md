# 組み込み

組み込みとは、OS のない小さなマイコンに直接コードを書き込む世界です。Raspberry Pi のように Linux が走るボードとは違い、起動した瞬間から自分のコードが動き始めます。

ここで使うのは手のひらに載る micro:bit v2。LED やセンサを動かしながら学んでいくうちに、BLE でスマホと通信できる段階まで来ます。スマートウォッチや体重計、スマートロックと同じ仕組みを自分のコードで動かせるようになれば、「この製品はどう動いているのか」がわかる側に立てます。それがこのガイドのゴールです。

「自分でデバイスを作りたい」という動機でも、「C や C++ で組み込みをやってきたが Rust でやり直したい」という動機でも、ここが入口になります。
