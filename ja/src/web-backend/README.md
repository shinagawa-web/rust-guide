# Web バックエンド

Rust で Web API を書く。Go や Python でバックエンドを書いてきた人が Rust に踏み込むための入口です。型システムが強く、動かして初めて気づくバグをコンパイル時に弾いてくれます。

このガイドでは axum を使います。既存の Web サーバーのコードを読んで直す、というスタイルで、最短で「Rust の Web アプリを変えた」体験を積めます。
