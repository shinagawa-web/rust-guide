# CLI ツール

自分用のコマンドラインツールを Rust で作る。単一のバイナリとして配れて、macOS 向け・Linux 向けそれぞれにビルドして届けられる。そういったツールを書く世界です。

シェルスクリプトの代替を探している人にも、Go や Python で CLI を書いてきて Rust でやってみたい人にも、入口として使えます。

## ガイド

- [初級](beginner.md) — ファイルの行数・文字数を表示する CLI ツールのバグを修正する。全 8 ページ・約 2 時間
- [中級](intermediate.md) — ツールにオプション・エラー処理・テストを加えて配布する。全 7 ページ・約 4 時間
