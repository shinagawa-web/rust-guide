# C から Rust へ

> 🚧 このセクションは現在準備中です。見出し構成のみ確定しています。

C 経験者向けに、Rust の基礎を一から順番に積む入門です。C との対比は理解を助ける補助として挟みます。

## この構成について

C 経験者なら「差分だけ拾い読みしたい」と感じるかもしれません。それでもこのセクションを差分の寄せ集めではなく、基礎から積む構成にしたのには理由があります。

- Rust の概念は所有権を中心に連鎖しています。借用・ライフタイム・エラー処理は所有権の上に積み上がるため、差分をつまみ食いすると土台が抜けて、かえって遠回りになります。
- だから順番に積む方が結局速い。所有権を理解するにはムーブの題材になる `String` が先に要るので、文字列を所有権の前に置いています。
- C との差分は各章の中で補助として添えます。「C ではこう、Rust ではこう」を要所で示し、手動でやっていたメモリ管理やヌルチェックがどう型に肩代わりされるかを足場にします。
- 章は独立ページに分割しています。必要なところだけ参照したり、詰まった概念に戻ったりしやすくするためです。
