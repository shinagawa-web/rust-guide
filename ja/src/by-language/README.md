# 言語別ガイド

すでに別の言語を使っている人向けの入口です。Rust の基礎を一から並べるのではなく、知っている言語との対比で読み進めます。同じ仕組みでも、馴染みのある言語のどれにあたるか、どこが違うかを起点にすると、抽象的なまま受け取るより理解しやすくなります。

## 言語から探す

- [Go から](from-go.md)：所有権・nil・interface・goroutine、Go とどう違うか
- [TypeScript から](from-typescript.md)：null 安全・エラー処理・trait・enum、TypeScript とどう違うか
- [C言語 から](from-c.md)：ポインタ・手動メモリ管理・未定義動作、C言語 とどう違うか
