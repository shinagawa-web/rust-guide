# diff をエッジから呼ぶ

この中級のはじめに、リクエストを受けてレスポンスを返す `fetch` ハンドラを手元で動かしました。前のページで、リクエストが裏で JavaScript を通って wasm の Rust に届く道すじを見ました。土台はもう整っています。

このページでやるのは、その `fetch` ハンドラに初級で手を入れたコードを差し込むことです。いまは決まった文字列を返すだけのハンドラを、差分を計算して返すハンドラに変えます。ただし、比べる2つのテキストは、このページではコードに直接書いておきます。送られてきたテキストを読み取るのは、次のページの仕事です。まずは、初級の diff がエッジの上で動くことを目で確かめます。

## ハンドラから diff を呼ぶ

開くのは、前に見た `src/lib.rs` です。いまはこうなっています。

```rust
#![allow(dead_code)]

use worker::*;

mod diff;

#[event(fetch)]
async fn fetch(_req: Request, _env: Env, _ctx: Context) -> Result<Response> {
    Response::ok("textdiff worker is running")
}
```

`Response::ok(...)` で決まった文字列を返しているだけの状態です。ここを、差分を計算して返すように書き換えます。

```rust
use worker::*;

mod diff;

#[event(fetch)]
async fn fetch(_req: Request, _env: Env, _ctx: Context) -> Result<Response> {
    let a = "I have a cat.";
    let b = "I have a dog.";
    Response::ok(diff::diff(a, b))
}
```

変えたのは、ハンドラの中身と、先頭から消した1行の2か所です。

比べる2つのテキストを `a` と `b` に置き、`diff::diff(a, b)` で差分を計算しています。`diff::diff` は、`mod diff;` で取り込んだ `src/diff.rs` の中の関数です。初級で文字化けを直した、あの文字ごとに差分を取るコードそのものです。返ってくるのは `String` で、消えた部分を `[-cat-]`、増えた部分を `[+dog+]` のように印で囲んだ文字列です。それをそのまま `Response::ok(...)` に渡して、レスポンスとして返します。

`a` と `b` をコードに直接書いているのは、このページではまだリクエストの中身を読まないからです。送られてきたテキストを取り出すには、リクエストの本文を解釈するもうひと手間が要ります。それは次のページでやります。ここでは、差分を計算するコードがエッジの上でちゃんと動くことだけに集中します。

先頭から消えた `#![allow(dead_code)]` は、前に「まだ使っていないコードへの警告を黙らせておくもの」と書いた1行です。いま `diff::diff` を呼んだことで、`mod diff;` で取り込んだコードは実際に使われるようになりました。警告は出なくなるので、黙らせるこの1行はもう要りません。役目を終えたので外します。

## 差分が返ってくるのを確かめる

書き換えたら、前と同じように手元で動かして確かめます。サンプルのフォルダで `wrangler dev` を起動します。

```sh
$ wrangler dev
```

書き換えたコードを worker-build が wasm に変換し直してから立ち上がるので、少し待ちます。起動したら、別のターミナルから同じ URL を叩きます。

```sh
$ curl http://localhost:8787
I have a [-cat-][+dog+].
```

これまでは `textdiff worker is running` という決まった文字列が返ってきました。今度返ってきたのは差分です。コードに書いた `I have a cat.` と `I have a dog.` の違いが、`[-cat-]` と `[+dog+]` の形で出ています。初級でブラウザの中で動かしたコードが、いまはエッジのランタイムの上で動いて、この結果を返しました。

返ってくる差分は、いまはコードに書いた `cat` と `dog` で決め打ちのままです。呼ぶ側が比べたいテキストを選べるようにするには、リクエストで送られてくるテキストをハンドラで読み取る必要があります。次のページでは、それを受け取って差分にして返せるようにします。
