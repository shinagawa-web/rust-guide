# Rust の基本構文

ツールが動くことは確認できました。次はコードを読んでいきます。ただし、いきなり題材のコードを開いても知らない構文が並んでいて読み進められません。ここでは、あとで出てくる書き方を先に押さえます。題材の差分ツールに実際に出てくるものだけに絞ります。

## let ― 変数を宣言する

変数を宣言するには `let` を使います。

```rust
fn main() {
    let n = 5;
    println!("{}", n);
}
```

Rust の変数は既定では書き換えられません。あとから値を変えたいときは `mut` を付けます。

```rust
fn main() {
    let mut count = 0;
    count = count + 1;
    println!("{}", count);
}
```

複数の変数をまとめて宣言することもできます。`let (a, b) = (1, 2);` は `a` に 1、`b` に 2 を入れます。

題材でも、組み立て中の差分の文字列などを `let mut` で持って、少しずつ書き換えていきます。

## 基本的な型

型はいくつも出てきますが、まず押さえておきたいのは次の 2 つです。

| 型 | 意味 |
|---|---|
| `u32` | 数値 |
| `String` | 文字列 |

## fn ― 関数を定義する

関数は `fn` で定義します。引数は `名前: 型`、戻り値の型は `->` の後に書きます。

```rust
fn add(a: u32, b: u32) -> u32 {
    a + b
}

fn main() {
    let sum = add(2, 3);
    println!("{}", sum);
}
```

Rust では最後の式がそのまま戻り値になるので、`return` を省略できます。上の `a + b` の行がそれです。戻り値がない関数は `->` ごと省略します。

題材では、2 つの文字列を受け取って差分を `String` で返す関数を中心に、処理をいくつかの関数に分けています。関数名の前に付く `pub` は、その関数を外から呼び出せるようにする指定で、4 章で扱います。

## if ― 条件によって分ける

条件で処理を変えるには `if` を使います。

```rust
fn main() {
    let n = 3;
    if n < 5 {
        println!("小さい");
    } else {
        println!("大きい");
    }
}
```

Rust の `if` は、値を返す式としても使えます。それぞれのブロックの最後の式がその値になり、`let` の右辺に直接置けます。

```rust
fn main() {
    let n = 3;
    let label = if n < 5 { "小さい" } else { "大きい" };
    println!("{}", label);
}
```

題材でも、条件によって入れる値を選ぶのに、この `if ... else` の形を使っています。

## for と while ― 繰り返す

決まった範囲を繰り返すには `for` を使います。`0..3` は 0・1・2 を表し、終わりの 3 は含みません。

```rust
fn main() {
    for i in 0..3 {
        println!("{}", i); // 0, 1, 2
    }
}
```

範囲のうしろに `.rev()` を付けると、逆順にたどれます。`.rev()` のように値のうしろに `.名前()` を付ける書き方は、その値に対する処理（メソッド）の呼び出しです。

```rust
fn main() {
    for i in (0..3).rev() {
        println!("{}", i); // 2, 1, 0
    }
}
```

条件が続くあいだ繰り返すには `while` を使います。

```rust
fn main() {
    let mut i = 0;
    while i < 3 {
        println!("{}", i);
        i += 1;
    }
}
```

`i += 1` は `i = i + 1` の短い書き方です。条件は `&&`（かつ）でつないで複数にすることもできます（`while i < 3 && j < 3` のように）。

題材でも、決まった回数の繰り返しに `(0..n).rev()` を、条件が続くあいだの繰り返しに `while` を使っています。

## struct ― 関連する値をまとめる

いくつかの値を 1 つにまとめた型を作るには `struct` を使います。

```rust
struct Point {
    x: u32,
    y: u32,
}

fn main() {
    let p = Point { x: 1, y: 2 };
    println!("{}", p.x);
}
```

`型名 { フィールド名: 値 }` で作り、`変数.フィールド名` で中身を読みます。題材でも、関連する値をまとめた型を、この `struct` で作っています。

## enum と match ― 種類で分ける

とりうる種類を並べて定義する型が `enum` です。

```rust
enum Direction {
    Up,
    Down,
}
```

`Direction::Up` のように `型名::種類` と書いて、その種類の値を指します。値がどの種類かで処理を分けるには `match` を使います。

```rust
enum Direction {
    Up,
    Down,
}

fn main() {
    let d = Direction::Up;
    match d {
        Direction::Up => println!("上"),
        Direction::Down => println!("下"),
    }
}
```

`match` は、その `enum` のすべての種類を書く必要があります。書き漏らすとコンパイルエラーになるので、種類の見落としを防げます。題材では、差分の各要素が「共通・消えた・増えた」のどれかを `enum` で表し、それを `match` で見て、消えた部分・増えた部分に囲み記号を付けています。

## Vec ― 同じ型の値を並べて持つ

同じ型の値を並べて持つには `Vec` を使います。`Vec<u32>` は「`u32` を要素に持つ `Vec`」という意味です。

```rust
fn main() {
    let mut xs: Vec<u32> = Vec::new();
    xs.push(10);
    xs.push(20);
    println!("{}", xs[0]); // 10
}
```

`push` で末尾に足し、`[添字]` で要素を読みます。添字は 0 始まりです。同じ値を並べて作りたいときは `vec!` が使えます。

```rust
fn main() {
    let zeros = vec![0u32; 3]; // 0 が 3 個
    println!("{}", zeros.len()); // 3
}
```

`vec![0u32; 3]` は「`0u32` を 3 個並べた `Vec`」、`.len()` は要素の個数です。題材でも、値を `Vec` に `push` で溜めたり、`vec![0u32; ...]` でまとめて用意したりしています。
