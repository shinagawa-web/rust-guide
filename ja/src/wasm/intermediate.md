# 中級

初級では、ブラウザの中で動く差分ツールを題材にしました。2つのテキストを比べて、消えた部分と増えた部分を色分けで表示する小さなツールです。そのコードを読み解き、日本語が文字化けする一か所を直しました。Rust がブラウザの中で動き、JavaScript から呼ばれる、という手応えをつかむ回でした。

中級で目指すのは、その差分ツールの Rust を、ブラウザの外へ持ち出すことです。同じ Rust のまま、世界中の拠点でコードを動かす「エッジ」に載せ、最後には誰でも呼び出せる URL として公開します。ブラウザの中でしか動かなかった処理が、HTTP で呼び出せる一つのサービスになります。

進め方は初級と同じです。まずローカルで動かして確かめ、そこから機能を足していき、最後に公開します。

## 出発点のリポジトリを用意する

中級では、専用のスターターリポジトリを出発点にします。初級で手を入れた、文字ごとに差分を取る Rust のコードだけを取り出し、それをエッジで動かすための最小のプログラムに収めたものです。初級で手を入れたコードとは別なので、混ざる心配はありません。

この先は Rust と cargo が入っている前提で進めます。まだ整えていない場合は、初級の [プロジェクトを動かす](beginner/run-project.md) で環境を用意してください。エッジで動かすための準備は、次の1章で整えます。

```sh
$ git clone https://github.com/shinagawa-web/rust-guide-sample-intermediate-wasm.git
$ cd rust-guide-sample-intermediate-wasm
```

## ゴール

中級を終えると、差分ツールはこう動きます。

2つのテキストを JSON で送ると、差分を JSON で返す HTTP エンドポイントになります。

```sh
$ curl -X POST https://textdiff-worker.<あなたのサブドメイン>.workers.dev/ \
    -H 'content-type: application/json' \
    -d '{"a": "I have a cat.", "b": "I have a dog."}'
```

```json
{"diff": "I have a [-cat-][+dog+]."}
```

この URL は Cloudflare のエッジに載っているので、世界のどこから呼んでも、近くの拠点が応えます。呼び出しているのは、初級から使ってきた、差分を計算する Rust のコードそのものです。ブラウザの中で動かしていた同じ処理が、今度は公開された URL の向こうで動きます。

呼び出す相手も curl だけではありません。小さな HTML のページから呼び出せば、ブラウザの入力欄に入れたテキストが、自分で公開したエッジの差分ツールを通って返ってきます。初級は差分の計算までブラウザの中で完結していましたが、中級ではその計算する場所が、ブラウザの外の、自分で公開したエッジに移ります。
