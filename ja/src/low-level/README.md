# 低レイヤ・システム

`chmod 755` の 755 が何を意味するのか。`Permission denied` がなぜ出るのか。`ls -l` の `-rwxr-xr-x` が何者なのか。こういう疑問の答えは OS の仕組みの中にあり、コードで手を入れられる層です。

Chrome や VS Code の下で動く仕組みも同じ層にあります。ふだんは見えませんが、自分で読み書きできるようになると、プログラムの見え方が変わります。

- [初級](beginner.md) — `ls -l` の左端と同じパーミッション文字列を表示するツールのバグを修正する。全 9 ページ・約 20,000 字・約 2 時間
